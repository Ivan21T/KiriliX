Инструкции за стартиране на програма написана на езика:
1) Windows
Отворя се Command Prompt или PowerShell в папката, където се намира kirilix.exe, и се изпълнява:
kirilix.exe път\до\файла.bg
Пример: Ако искате да стартирате примерния файл 03_control_flow.bg:

kirilix.exe examples\03_control_flow.bg
2)Linux/MacOs
Отворяте терминала (Terminal) в папката, където се намира компилираният файл kirilix. Първо се уверявате, че файлът има права за изпълнение.
Команда за стартиране: ./kirilix път/към/файла.bg
Пример: ./kirilix examples/03_control_flow.bg
