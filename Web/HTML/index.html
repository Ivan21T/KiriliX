Интуитивен синтаксис с български думи

KiriliX използва български ключови думи, които са лесни за запомняне и правят кода по-четим за българските програмисти.

  • Ключови думи като "функция", "ако", "за", "докато"
  • Типове данни: "цяло", "дробно", "текст", "символ", "булев"
  • Функции за вход/изход: "печатай", "прочети"
  • Съобщения за грешки на български език
български-пример.bg
// Изчисляване на факториел
функция цяло факториел(цяло число) {
  ако (число <= 1) {
    върни 1;
  }
  върни число * факториел(число - 1);
}
// Главна функция
функция празно главна() {
  цяло резултат = факториел(5);
  печатай(резултат);
  върни нищо;
}
